## Installation

1. First, clone the repository:

    ```bash
    git clone https://github.com/sayenvv/social-media-platform.git
    ```
2. Second,checkout to development branch
   ```bash
    git checkout develop
    ```

2. Install dependencies:

    ```bash
    cd social_networking_media
    docker compose buld
    docker compose up
    ```
